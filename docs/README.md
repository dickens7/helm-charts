# Helm Charts

> Prerequisites
> - Kubernetes cluster 1.18+
> - Helm 3.1.0+


## add a chart repository

```bash
helm repo add dickens7 https://dickens7.github.io/helm-charts/
```

## update information of available charts locally from chart repositories

```bash
helm repo update
```
