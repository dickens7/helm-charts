# etcdkeeper

> Prerequisites
> - Kubernetes cluster 1.18+
> - Helm 3.0.0+

## Quick Start

- Install the chart

```shell
helm install <name> dickens7/etcdkeeper . -n <namespace> 
```

- Uninstallation

```
helm uninstall <name> -n <namespace> 
```
